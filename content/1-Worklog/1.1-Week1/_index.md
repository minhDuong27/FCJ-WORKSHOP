### Week 1 Achievements:

* Gained a clear understanding of AWS and the key service groups:
  * Compute  
  * Storage  
  * Networking  
  * Database  
  * Security, Identity & Compliance  
  * Analytics  
  * Machine Learning & AI  
  * Management & Monitoring  

* Successfully signed up for and configured a personal AWS Free Tier account.  
* Became comfortable using the AWS Console to locate and interact with different services.  
* Installed and configured AWS CLI, including:
  * Access Key setup  
  * Secret Key setup  
  * Choosing a Default Region  

* Practiced performing essential operations via AWS CLI, such as:
  * Checking account details and IAM identity.  
  * Managing S3: create, view, upload, and download files.  
  * Working with EC2: create, start, stop, and describe instances.  
  * Using RDS: list and describe available databases.  
  * Managing IAM: add new users, assign groups, and set permissions.  
  * Viewing VPC configurations such as subnets and security groups.  
  * Monitoring logs and metrics using CloudWatch.  
  * Managing DNS records and hosted zones in Route 53.  
  * Deploying and invoking Lambda functions.  
  * Creating and managing CloudFormation stacks.
